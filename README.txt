 git add . chuyen tat ca thu muc dc thay doi vao trang thai san sang commit
 git add README.txt chi chuyen thu muc README vao trang thai san sang commit

 git commit -m "text" thuc hien 1 commit va dat ten

git status kiem tra trang thai git hien tai

git tag <ten tag> <hash> dat tag cho 1 commit

git branch thao tac voi cac nhanh
 git branch -l liet ke cac nhanh
